# 🚀 Portfólio Letícia Moreira - MCODE

Este é o meu portfólio profissional, desenvolvido para apresentar meus serviços como Desenvolvedora Frontend e UI Designer. O projeto foca em uma estética moderna, performance otimizada e total responsividade para dispositivos móveis.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** - Estrutura semântica.
* **CSS3** - Estilização avançada com Grid, Flexbox e Media Queries.
* **JavaScript (Vanilla)** - Lógica de interatividade e animações de scroll.
* **Font Awesome** - Biblioteca de ícones.
* **Google Fonts** - Tipografia personalizada (Poppins).
* **Netlify** - Hospedagem e deploy contínuo.

---

## 📱 Funcionalidades

- [x] **Design Responsivo:** Adaptado para Desktop, Tablets e Smartphones.
- [x] **Header Dinâmico:** Efeito de transparência ao rolar a página (Blur).
- [x] **Navegação Suave:** Âncoras internas para facilitar a experiência do usuário.
- [x] **Botão de WhatsApp:** Link direto para orçamento configurado dinamicamente via JS.
- [x] **Efeito Visual Premium:** Texto com fundo de imagem vazado no título principal.

---

## 🎨 Layout

O layout foi planejado para transmitir profissionalismo e modernidade, utilizando um esquema de cores *Dark Mode* com detalhes em roxo suave (`#a29bfe`) para destacar elementos de ação.
